ABOUT THE iSTATS

This website allow dimensionality reduction which refers to techniques that reduce the number of input variables in a dataset. More input features often make a predictive modeling task more challenging to model, more generally referred to as the curse of dimensionality.

Techniques offered by iSTATS

  1. Isomap
  2. Sparse Random Projection
  3. Gaussian Random Projection
  4. Multi Dimension Scaling
  5. Isomap
  6. T-sne
