STEPS TO RUN:
1.install django 
2.Download the rndframework folder from git repository
3.Install the libraries mentioned in views.py file in calc folder
4.Go to terminal 
5.Set path to manage.py location 
6.Now run command "python manage.py runserver"
7.Enjoy iSTATS website by clicking on the localhost link generated  "http://127.0.0.1:8000/"
